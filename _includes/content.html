Projects

  • 2013-2014 - Design and evaluation of a pipelined network interface for network on chip applications. A project of ESRC Laboratory.
  • 2012-2013 - Implementation of a 2-by-2 Torus Router for Network-on-Chip. A project of ESRC Laboratory. In this project, a 2x2 Torus network is implemented as a result of our group's literature research on Network-on-Chip. Torus topology, with simplex links, is used as it is very simple and efficient for small network. The deadlock-free network is implemented by using Virtual Channel technique and Wormhole switching. The design is written in VHDL and implemented on Spartan 3 FPGA board.
  • 2012-2012 - Traffic classification with optical flow. A project of ESRC Laboratory. The objective is to develop an embedded system on trafic camera that can classify current trafic situation. My team of 3 worked on a PC software that takes video files as input and return a classification from A to F. The software is developed in Visual Studio 2010 C++ environment with OpenCV library.
